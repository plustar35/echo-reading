# 更新日志

本项目所有值得记录的变更都写在这里。全程中文，按日期倒序。

## 2026-05-30

### 升级：溯源链格式 `[[书名:章节]]` → `[[书名/chNN/单元|显示名]]`

把贯穿全项目的引用格式从「书→章」升级成「书→章→单元」，并默认指向 `raw.md` 底本，在 Obsidian 里可一路点回原文。

- **chapter-split**（`scripts/write_units.py`）：生成的阅读单元、`00-导读.md` 首行溯源链改用新格式，指向 `[[书名/chNN/raw|《书名》章名]]`。
- **已切书稿回填**：《理想国》ch02 全部单元 + 导读、《道德经》ch01/ch02 单元首行溯源链统一改成新格式。
- **insight/README.md**：引用约定更新为 `[[书名/章节/单元|显示名]]`，明确指向单元文件 `chNN/NN.md`；维度说明表重排。
- **insight/INDEX.md**：骨架行的来源格式同步更新；清空「你的故事」「闪回」下的示例条目，回到「暂无条目」干净状态。

### 重构：CLAUDE.md 操作规则

- **反链不变量**拆成两类：**溯源链**（单向，衍生文件 → 来源，靠每个文件首行「来自」串起）+ **概念双向链**（A 写 `[[B]]`，B 文件底部 `## 提及位置` 登记 A）。
- 新增不变量：**每个 insight 文件都有 frontmatter**（`title` / `sources` / `tags` / `updated`）。
- 工作流「读《X》第 N 章」的第 0 步与分诊逻辑精简；insight 写入判断的措辞调整。

### 新增：AGENTS.md + `.agents` 符号链接

- 新增 `AGENTS.md`（= `CLAUDE.md` 镜像），让遵循 AGENTS 约定的其他 agent 工具读到同一套规则。
- 新增 `.agents` → `.claude` 符号链接，复用既有的 skills / 配置目录。
